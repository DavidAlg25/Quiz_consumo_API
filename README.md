# Proyecto Login Flutter - Evaluación

Aplicación móvil desarrollada en **Flutter** que se conecta a una **API REST** creada con **Node.js** y **MySQL**. Permite registrar nuevos usuarios, iniciar sesión y acceder a una pantalla de inicio con contenido aleatorio.

---

##  Requisitos previos

Asegúrate de tener instalado lo siguiente:

- **Git** – para clonar el repositorio.
- **Visual Studio Code** (u otro editor) – recomendado.
- **Node.js** (última versión estable) – para el backend.
- **Flutter SDK** con **Dart SDK** – para el frontend.
- **MySQL** (se recomienda **XAMPP** para tener MySQL y phpMyAdmin).

---
